TEEHR Documentation#

TEEHR (pronounced “tier”) is a Python tool set for loading, storing, processing and visualizing hydrologic data, particularly National Water Model data, for the purpose of exploring and evaluating the datasets to assess their skill and performance.

Getting Started

New to TEEHR? Check out the getting started guides. They contain an introduction to TEEHR main concepts and links to additional tutorials.

API Reference

The reference guide contains a detailed description of the TEEHR API. The reference describes how the methods work and which parameters can be used. It assumes that you have an understanding of the key concepts.
