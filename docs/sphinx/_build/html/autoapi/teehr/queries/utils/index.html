teehr.queries.utils.df_to_gdf(df: pandas.DataFrame) → geopandas.GeoDataFrame[source]#

Convert pd.DataFrame to gpd.GeoDataFrame.

When the geometry column is read from a parquet file using DuckBD it is a bytearray in the resulting pd.DataFrame. The geometry needs to be convert to bytes before GeoPandas can work with it. This function does that.

Parameters:
dfpd.DataFrame

DataFrame with a geometry column that has geometry stored as a bytearray.

Returns:
gpd.GeoDataFrame

GeoDataFrame with a valid geometry column.
