__pydantic_extra__: An instance attribute with the values of extra fields from validation when

model_config[‘extra’] == ‘allow’.

__pydantic_fields_set__: An instance attribute with the names of fields explicitly set. __pydantic_private__: Instance attribute with the values of private attributes set on the model instance.
