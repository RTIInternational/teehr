import_locations(in_path: pathlib.Path | str, field_mapping: dict = None, pattern: str = '**/*.parquet', **kwargs)#

Import geometry data.

Parameters:
in_pathUnion[Path, str]

The input file or directory path. Any file format that can be read by GeoPandas.

field_mappingdict, optional

A dictionary mapping input fields to output fields. Format: {input_field: output_field}

patternstr, optional (default: “**/*.parquet”)

The pattern to match files. Only used when in_path is a directory.

**kwargs

Additional keyword arguments are passed to GeoPandas read_file().

File is first read by GeoPandas, field names renamed and
then validated and inserted into the dataset.
The TEEHR Location Crosswalk table schema includes fields:
- id
- name
- geometry
